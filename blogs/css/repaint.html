    回流重绘

    漫思维 CSS

    # 产生原因

    浏览器在渲染页面时会将 HTML 和 CSS 分别解析成 DOM 树和 CSSOM 树，然后合并进行排布，再绘制成我们可见的页面。如果在操作 DOM 时涉及到元素、样式的修改，就会引起渲染引擎重新计算样式生成 CSSOM 树，同时还有可能触发对元素的重新排布（简称“重排”）和重新绘制（简称“重绘”）。

    # 回流

    触发条件：

    • 修改元素边距、大小

    • 添加、删除元素

    • 改变窗口大小

    # 重绘

    • 设置背景图片
    • 修改字体颜色
    • 改变Visibility属性值
